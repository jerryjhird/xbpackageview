# xbpackageview

## allows you to pull metadata from stfs xbox360 packages this includes:

PACKAGE TYPES:
 
 LIVE

 CON
 
 PIRS

CONTENT TYPES:

 SAVE GAME

 DLC

 PROFILE

 ARACADE GAME
 
 TITLE UPDATE


## you can currently get the following information

package type: [LIVE, CON, PIRS]

content type: [DLC, PROFILE, SAVEGAME, ETC]

title id

content id

package name [the name of the game, dlc etc]

title name

package description [game or dlc's description]

publisher name
